Calculator

	This project is writen just for practise. It's a simple calculator

application. This calculator implemented functions below:
	+                         加法
	-                         减法
	*                         乘法
	/                         除法
	^                         乘方
	%                         取模运算
	!                         阶乘
	=                         判等
	&                         按位与运算
	|                         按位或运算
	~                         按位取反
	&&                        逻辑与运算
	||                        逻辑或运算
	!                         逻辑非运算
	()                        括号
	fib(n)                    计算第 n 个斐波那契数
	sqrt(n)                   计算 n 的平方根
	cbrt(n)                   计算 n 的立方根
